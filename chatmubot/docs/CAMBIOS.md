# Cambios

## Funcionales
- Adivinanza con estado real, pistas y marcador.
- Duelo con validación más robusta y soporte de palabras clave.
- Contexto con historial de temas recientes.
- Comandos locales de memoria, exportación y reinicio.

## Estructurales
- Separación clara entre core, módulos, datos, documentación y herramientas.
- Configuración centralizada.
- Loader desacoplado de rutas frágiles.
- Conservación del dataset original y generación de dataset runtime.

## Mantenibilidad
- Menos rutas duplicadas.
- Mejor punto de entrada.
- Menor acoplamiento entre HTML y rutas internas.
